    เอกสารการทดลอง

    สำหรับใช้กับชุดทดลอง ESP32 Kit1

    เป้าหมายการเรียนรู้

    การทดลองต่อไปนี้เป็นตัวอย่างในการใช้ ESP32 เพื่ออ่านค่าจากเซนเซอร์และนำค่าที่ได้แสดงในโทรศัพท์มือถือ หรือ เข้าดูผ่านเว็บไซต์ ทำให้ผู้ใช้งานสามารถอ่านค่าเซนเซอร์ที่ติดตั้งไว้ จากที่ใดก็ได้ ตราบเท่าที่สามารถเข้าถึงระบบอินเทอร์เน็ต (ทั้งฝั่งส่งและรับ)

    นอกจากนี้สามารถประยุกต์ใช้ในการควบคุมอุปกรณ์ระยะไกลได้ เพื่อให้เกิดความสมบูรณ์ในการแสดงค่า การทดลองต่อไปนี้จึงสาธิตการแสดงผลการวัดที่ serial monitor และ TFT LCD อีกด้วย

    เนื้อหาบทเรียน

    บทที่ 1: LED ไฟกะพริบ

    สำหรับบอร์ด NodeMCU ESP32 ที่ใช้นี้มี Light Emitting Diode: LED ที่ติดตั้งอยู่บนตัวบอร์ดไว้ให้แล้ว (On board LED)

    การติดสว่างของ LED นั้นเกิดขึนเมื่อจ่ายแรงดันทำงาน 3V ให้กับ LED และเมื่อหยุดจ่ายแรงดัน 3V ก็จะหยุดเปล่งแสง โดยสามารถทดสอบด้วย Sketch ดังต่อไปนี้
    //LED Blink
//The LED will turn on for one second and then turn off for one second
int ledPin = 12;
void setup() {
    pinMode(ledPin, OUTPUT);
}
void loop() {
    digitalWrite(ledPin, HIGH);
    delay(1000);
    digitalWrite(ledPin, LOW);
    delay(1000);
}

    รู้หรือไม่ การทดลองไฟกะพริบนั้นเป็นจุดเริ่มต้นในการใช้งาน นอกจากนี้หากสงสัยว่าตัวอุปกรณ์มีปัญหาในการทำงานนั้น ก็ยังสามารถนำมาใช้ในการทดสอบการทำงานเบื้องต้นของตัว NodeMCU ได้
